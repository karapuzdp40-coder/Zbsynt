## 🎯 Как установить на телефон

### **На Android телефоне (напрямую)**

1. **Включите Developer Mode:**
   - Settings → About Phone
   - Нажмите "Build Number" 7 раз
   - Появится "You are now a developer!"

2. **Включите USB Debugging:**
   - Settings → Developer Options
   - Включите "USB Debugging"

3. **Подключитесь к ПК:**
   - Подключите телефон через USB кабель
   - Нажмите "Allow" на телефоне для отладки

4. **Установите APK:**
   ```bash
   adb install app-debug.apk
   ```

### **Без ПК (напрямую на телефон)**

1. Скопируйте файл `app-debug.apk` на телефон
2. Откройте файловый менеджер
3. Найдите файл и нажмите на него
4. Нажмите "Установить"
5. Разрешите установку из неизвестных источников

## ⚠️ Решение проблем

### APK не устанавливается

```bash
# 1. Удалите старую версию
adb uninstall com.absynth.hostel

# 2. Переустановите с флагом -r
adb install -r app-debug.apk
```

### "Неизвестный источник" ошибка

1. Settings → Security
2. Включите "Unknown Sources" (для вашей версии Android)
3. Попробуйте установить снова

### Приложение вылетает при запуске

1. Удалите приложение
2. Перезагрузитесь
3. Переустановите APK
