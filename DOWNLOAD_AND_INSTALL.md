## ⚠️ **РЕШЕНИЕ ПРОБЛЕМ**

### ❌ **"Неизвестный источник" ошибка**

**На Android 10+:**
1. Settings → Apps & notifications
2. Advanced → Install unknown apps
3. Выберите ваш файловый менеджер
4. Включите "Allow from this source"

**На Android 9 и ниже:**
1. Settings → Security
2. Включите "Unknown Sources"

### ❌ **"App not installed" ошибка**

```bash
# Удалите старую версию:
adb uninstall com.absynth.hostel

# Переустановите:
adb install -r app-debug.apk
```

### ❌ **Приложение вылетает при запуске**

1. Удалите приложение полностью
2. Перезагрузите телефон
3. Переустановите APK
4. Очистите кэш приложения

### ❌ **"Недостаточно памяти" ошибка**

1. Удалите ненужные приложения
2. Очистите кэш системы
3. Попробуйте снова
